# 8085-Assembly-program
A collection of Intel 8085 Assembly language programs including manual hex-code mapping, register operations, and logic verification using both hardware trainer kits and mobile simulators.
### 🛠️ Key Technical Details
* **Mnemonics & Hex Mapping:** Manual translation of assembly instructions into hex codes for memory entry.
* **Architecture Focus:** Register-level programming, addressing modes, and flag register monitoring.
* **Environment:** Implemented on 8085 Trainer Kits and verified via mobile simulation.

### 📂 Featured Programs
* **Memory Management:** Programs to load, store, and manipulate data at specific memory addresses (e.g., 2050H, 2100H).
* **Arithmetic Logic:** Subtraction and addition with carry handling.
